📖 使用示例

1. 获取随机图片信息 (JSON)
GET /api/random GET /api/random?user_id=1 GET /api/random?tags=风景
2. 直接获取随机图片文件
<img src="http://localhost:8080/api/random/image" alt="随机图片"> <img src="http://localhost:8080/api/random/image?tags=风景" alt="风景图片">
3. 重定向到随机图片
GET /api/random/redirect GET /api/random/redirect?user_id=1&tags=自然
4. JavaScript 示例
fetch('/api/random?tags=风景') .then(res => res.json()) .then(data => { console.log('随机图片:', data); const imageUrl = `/i/${data.uuid}`; });
5. 作为网站背景
<div style="background-image: url('http://localhost:8080/api/random/image?tags=壁纸')"> </div>
